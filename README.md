# Calculadora Web

Uma calculadora moderna desenvolvida com tecnologias web padrão.
O foco deste projeto foi criar uma interface limpa, responsiva e com feedback visual tátil (neumorphism ou flat design) para o usuário.

## Funcionalidades

- Operações matemáticas básicas (+, -, *, /).
- Histórico de cálculos recentes.
- Tratamento de erros (ex: divisão por zero).
- Design responsivo (funciona em mobile).

## Stack

- HTML5
- CSS3
- JavaScript

## Como rodar

Abra o arquivo `calculadora.html` no seu navegador.
